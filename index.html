What is phpMussel?

An ideal solution for shared hosting environments, where it's often not possible to utilise or install conventional anti-virus protection solutions, phpMussel is a PHP script designed to detect trojans, viruses, malware and other threats within files uploaded to your system wherever the script is hooked, based on the signatures of ClamAV and others. For information regarding HOW TO INSTALL {2A+2B} and HOW TO USE {3A+3B} phpMussel, please refer either to the Wiki or to documentation included within the "_docs" directory of this repository (direct links to that documentation included under the "Documentation" header below this paragraph).

Features:

  • Licensed as GNU General Public License version 2.0 (GPLv2).
  • Easy to install, easy to customise, easy to use.
  • Works for any system with PHP+PCRE installed, regardless of OS (PHP+PCRE required).
  • Fully configurable based on your needs.
  • Ideal solution for shared hosting services.
  • Ideal solution for forum systems in need of file upload protection.
  • Does NOT require shell access.
  • Does NOT require administrative privileges.
  • CLI mode available (for now, just under Windows, very soon with other OS).
  • Good, strong, stable support base.

Want to donate?

I don't expect any financial remuneration, compensation nor reward for any work done for phpMussel, nor are you obliged to offer any; phpMussel is provided absolutely free of charge and will indefinitely stay that way. However, if you -want- to offer something in the way of a donation, you're able to do so by using the PayPal donate button provided to your left, under the download links provided on this page.

I haven't yet received any such donations, but if I do, if they amount to anything significant, they'll mostly likely go towards reimbursing any already incurred costs for the project, financing the project for further improvements and possibly forwarding on some of that amount to the various other services that've contributed to phpMussel's development, and if I receive any such donations but they don't amount to much, they'll at least go towards making me feel good about the project and possibly cover the costs of late night coffees and such things. :-)
